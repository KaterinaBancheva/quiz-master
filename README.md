# QuizMaster 🎯

Конзолно приложение за създаване и решаване на quiz-ове, вдъхновено от Kahoot, Quizizz и др.

## Функционалности

### Типове въпроси
- **TrueOrFalseQuestion** — въпрос с два възможни отговора (истина/лъжа)
- **SingleChoiceQuestion** — въпрос с четири възможни отговора, само един верен
- **MultipleChoiceQuestion** — въпрос с поне четири възможни отговора, няколко верни
- **ShortAnswerQuestion** — въпрос с кратка фраза или числова стойност като отговор
- **MatchingPairsQuestion** — въпрос с две колони, които трябва да се свържат

### Режими на работа
- **Test режим** — показва верните отговори след всеки въпрос, не се трупат точки
- **Normal режим** — не показва верните отговори, трупат се точки
- **Shuffle опция** — разбърква въпросите в произволен ред

### Потребители
- **Администратор** — управлява quiz-ове, разглежда доклади, блокира потребители
- **Играч** — създава и решава quiz-ове, трупа точки и изпълнява предизвикателства

### Точки и нива
| Ниво | Точки |
|------|-------|
| 1–10 | 1000 точки |
| 11–20 | 2000 точки |
| 21–30 | 3000 точки |
| 31+ | 4000 точки |

### Предизвикателства
- Решаване на N quiz-а в тестов режим
- Решаване на N quiz-а в нормален режим
- Създаване на N quiz-а
